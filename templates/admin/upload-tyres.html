📋 CSV Format Guide

Your CSV file should have the following columns in this order:

brand,size,condition,tread,price,description

Note: The first row should be column headers. Make sure the data matches the column order exactly.

📝 Column Details

  • brand: Tyre brand name (e.g., Michelin)
  • size: Tyre size (e.g., 205/55R16)
  • condition: Condition description (e.g., Good, Excellent)
  • tread: Tread depth (e.g., 7/32", 8mm)
  • price: Price in dollars (e.g., 45.99)
  • description: Additional details (optional)

💡 Example CSV Content

brand,size,condition,tread,price,description
Michelin,205/55R16,Excellent,8mm,65.00,Low mileage
Goodyear,225/60R17,Good,6/32in,45.50,Well maintained
Bridgestone,195/65R15,Fair,4mm,35.00,Some signs of wear
Continental,215/55R17,Excellent,7mm,72.50,Recently imported
Pirelli,205/60R16,Good,5mm,48.75,Good grip remaining

⚙️ Instructions

  1. Prepare your CSV file with the proper format (see example above)
  2. Make sure all required columns are present: brand, size, condition, tread, price, description
  3. Save the file as a CSV (Comma-Separated Values) file
  4. Click the upload area or drag your file to upload
  5. The system will process the file and display the tyres on the Used Tyres page
  6. You can upload a new file at any time to update the inventory
